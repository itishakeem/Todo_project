## Mandatory Features (non-negotiable)

### Integrated AI Chatbot (Phases III, IV, V)
The application **must** provide a real-time conversational interface capable of managing the entire Todo list via natural language using **only** the following official SDKs:
- OpenAI Chatkit
- OpenAI Agents SDK
- Official OpenAI MCP SDK (Multi-Modal Control Plane for persistent memory)

Example supported commands:
- “Add buy groceries tomorrow at 6pm”
- “Reschedule my morning meetings to 2pm”
- “Mark the report as high priority”
- “What do I have this week?”

Implementation MUST be generated by Claude Code using the prompt template inside `ai-chatbot.spec.md`.

### Cloud Native Deployment (Phases IV & V)
The entire stack (frontend, API, AI agent, database, Redis) **must** run:
- Locally on Minikube (`make k8s-up`)
- In production on DigitalOcean Kubernetes (DOKS) via GitHub Actions + Helm

Zero-downtime upgrades, autoscaling, cert-manager + Let’s Encrypt, and Prometheus/Grafana are required.

Implementation MUST be generated by Claude Code using the prompt template inside `kubernetes-deployment.spec.md`.
