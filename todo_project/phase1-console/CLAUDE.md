# Claude Code - Phase 1: Console Application

This file contains project-specific instructions for Claude Code related to Phase 1.

## Task Management Logic

This phase focuses on building the core task management logic for the todo application. This includes:

- Adding new tasks
- Deleting tasks
- Updating existing tasks
- Listing all tasks
- Marking tasks as complete

## Development Guidelines

- Implement robust error handling for all operations.
- Write comprehensive unit tests for all core logic.
- Ensure the console interface is user-friendly and provides clear feedback.
