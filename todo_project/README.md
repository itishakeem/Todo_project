# Todo Evolution Project

This project aims to evolve a todo list application through multiple phases, incorporating various technologies and architectural patterns.

## Phases

- **Phase 1: Console Application** - Core task management logic.
- **Phase 2: Web Application** - Frontend and backend for a user-facing interface.
- **Phase 3: AI Integration** - Incorporating AI agents and chatbot functionality.
- **Phase 4: Containerization** - Deploying with Kubernetes and Dapr.
- **Phase 5: Cloud Deployment** - CI/CD, monitoring, and cloud infrastructure.
