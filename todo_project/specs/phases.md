## Phase 2: Web Application
- Frontend and backend for a user-facing interface.

- Incorporating AI agents and chatbot functionality.

## Phase 4: Containerization
- Deploying with Kubernetes and Dapr.

- CI/CD, monitoring, and cloud infrastructure.
