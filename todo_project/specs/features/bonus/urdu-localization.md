# Feature: Urdu Localization

## Description

This feature will add support for the Urdu language, allowing users to interact with the application in Urdu.

## Goals

- Localize all user-facing text and UI elements into Urdu.
- Ensure correct display of Urdu script (right-to-left).

## Acceptance Criteria

- The application is fully translated into Urdu.
- Urdu text is displayed correctly, respecting right-to-left rendering.
